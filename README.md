# 🏠 House Price Prediction using Machine Learning

A web-based machine learning project that predicts house prices based on features such as area, number of bedrooms, bathrooms, property age, location, and distance from the city center.

Built using **Python, Pandas, Scikit-learn, and Streamlit**, the project demonstrates an end-to-end machine learning workflow from data preprocessing to model prediction.

## 📌 Features

### 📊 Data Preprocessing

* Load and clean housing datasets
* Handle missing values and duplicate records
* Encode categorical variables
* Prepare data for machine learning models

### 📈 Exploratory Data Analysis (EDA)

* Visualize feature distributions
* Analyze relationships between variables
* Generate correlation matrices
* Identify important factors affecting house prices

### 🤖 Machine Learning Model

* Train a regression model for price prediction
* Split data into training and testing sets
* Evaluate model performance
* Predict prices for new house details

### 🖥 Web Application

* User-friendly Streamlit interface
* Input house details interactively
* Display predicted house price instantly
* Easy-to-use dashboard for demonstrations

## 🔧 Installation & Setup

Create a virtual environment and install dependencies:

```bash
python -m venv env

# Windows
env\Scripts\activate

# Linux / Mac
source env/bin/activate

pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
